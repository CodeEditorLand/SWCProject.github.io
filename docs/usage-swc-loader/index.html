Using swc with webpack

Installation#

npm i --save-dev @swc/core swc-loader

Configuration#

module: {  rules: [    {      test: /\.m?js$/,      exclude: /(node_modules|bower_components)/,      use: {        // `.swcrc` can be used to configure swc        loader: "swc-loader"      }    }  ];}
